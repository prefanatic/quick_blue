# quick_blue_platform_interface

The shared platform interface, models, and device/GATT abstractions for the
[`quick_blue`](https://pub.dev/packages/quick_blue) federated Flutter plugin.

Applications should depend on `quick_blue` instead. Platform implementations
use this package to integrate with the app-facing API.

See the [QuickBlue repository](https://github.com/prefanatic/quick_blue) for API,
implementation, and development documentation.
